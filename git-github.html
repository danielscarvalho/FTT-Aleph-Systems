Git & GitHub passo a passo

No site do GitHub criar um novo repositório: www.github.com

No diretório local do seu projeto WEB (ou Java, Python, etc) executar os comandos (no seu PC):

1. Criar um arquivo texto markdown (.md) para detalhar seu projeto: 

echo "# FTT-Aleph-Systems" >> README.md

2. Iniciar o git no seu projeto, este comando cria o diretório oculto .git no diretório do projeto 

git init

3. Adicionar os arquivos do seu projeto no git (HEAD) 

git add README.md
git add *

4. Fechar um conjunto de modificações 

git commit -m "first commit"

5. Estabelecer qual é o repositório no GitHub que será utilizado (isso é necessário definir apenas uma vez) 

git remote add origin https://github.com/danielscarvalho/FTT-Aleph-Systems.git

6. Se estiver em uma rede com proxy, incluir o parâmetro 

git config --global http.proxy http://<ra>:<senha>@<PROXY IP>:8080

7. Enviar o código para o GitHub (Nuvem) 

git push -u origin master

No dia a dia, depois do setup inicial, usar a sequencia de comando abaixo para carregar as alterações para o servidor do Github em nuvem:

git add *
git commit -m "you know"
git push

Para atualizar o projeto do servidor para o ambiente local use:

git pull

Para fazer um novo ambiente a partir do GitHub:

git clone https://github.com/danielscarvalho/FTT-Aleph-Systems.git

A qualquer momento você pode verificar o status do git:

git status
Para publicar sua página WEB estática com arquivos HTML5 e CSS3, arquivos de imagem, txt, etc...
No seu repositório do GitHub selecione o menu Settings vá para o final da página na sessão GitHub Pages e em Source selecione master branch.

O link para este projeto de exemplo é: https://danielscarvalho.github.io/FTT-Aleph-Systems/
Lembre-se que o arquivo principal de um projeto WEB deve ser nomeado como index.html

Para compartilhar seu projeto com a equipe selecione o menu Settings, Colaborators e adicione os membros do seu grupo...

Se você meteu a pata e detonou tudo! Não entre em pânico! Remova a pasta .git no diretório do seu projeto e faça tudo dinovo!
