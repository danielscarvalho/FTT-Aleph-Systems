Git & GitHub passo a passo

No site do GitHub criar um novo repositório: www.github.com No diretório do seu projeto WEB (ou Java, Python, etc) executar os comandos: 1. Criar um arquivo texto markdown (.md) para detalhar seu projeto: 
echo "# FTT-Aleph-Systems" >> README.md
2. Iniciar o git no seu projeto, este comando cria o diretório oculto .git no diretório do projeto 
git init
3. Adicionar os arquivos do seu projeto no git (HEAD) 
git add README.md
git add *
4. Fechar um conjunto de modificações 
git commit -m "first commit"
5. Estabelecer qual é o repositório no GitHub que será utilizado (isso é necessário apenas definir apenas uma vez) 
git remote add origin https://github.com/danielscarvalho/FTT-Aleph-Systems.git
6. Se estiver em uma rede com proxy, incluir os parâmetros 
git config --global http.proxy http://:@:8080
7. Enviar o código para o GitHub (Nuvem) 
git push -u origin master
Para publicar sua página WEB estática com arquivos HTML5 e CSS3, arquivos de imagem, txt, etc... No seu repositório do GitHub selecione o menu Settings vá para o final da página na sessão

GitHub Pages

e em Source selecione master branch.
Para compartilhar seu projeto com a equipe selecione o menu Settings, Colaborators e adicione os usuários...
